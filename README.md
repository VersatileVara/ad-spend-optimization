# Ad Spend Optimization
### Minor Project — Marketing Analytics | Data Analytics Portfolio

## Objective
Analyze multi-channel advertising spend data to identify the most 
profitable ad channels and optimize budget allocation.

## Tools Used
Python, Pandas, Matplotlib, Seaborn, Power BI

## Dataset
Marketing Spend Dataset — Kaggle (308 rows, 11 columns)

## Key Findings
- youtube_blogger: ROI 307%, ROAS 4x — Best performer
- facebook_retargeting: ROI 145% — Hidden gem
- facebook_lal: ROI -88% — Immediate reallocation needed

## Project Structure
- analysis.py — Python analysis code
- cleaned_ad_spend.csv — Processed dataset
- *.png — Visualization charts
